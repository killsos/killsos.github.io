2. Nullish Coalescing Operator ?? undefined or null

Base case. If the expression at the left-hand side of the ?? operator evaluates to undefined or null, its right-hand side is returned.

const response = {
    settings: {
        nullValue: null,
        height: 400,
        animationDuration: 0,
        headerText: '',
        showSplashScreen: false
    }
    };

    const undefinedValue = response.settings.undefinedValue ?? 'some other default'; // result: 'some other default'
    const nullValue = response.settings.nullValue ?? 'some other default'; // result: 'some other default'
    const headerText = response.settings.headerText ?? 'Hello, world!'; // result: ''
    const animationDuration = response.settings.animationDuration ?? 300; // result: 0
    const showSplashScreen = response.settings.showSplashScreen ?? true; // result: false

3. BigInt: Arbitrary Precision Integers in JavaScript 高精度整型

IEEE754 64-bit floats  BigInt 可以表示 2^53

63 符号位

62-53 指数位

0-52 精度位

const x = Number.MAX_SAFE_INTEGER;
// ↪ 9007199254740991, this is 1 less than 2^53

const y = x + 1;
// ↪ 9007199254740992, ok, checks out

const z = x + 2
// ↪ 9007199254740992, wait, that’s the same as above!

A BigInt is created by appending n to the end of the integer or by calling the constructor:

const theBiggestInt = 9007199254740991n;

const alsoHuge = BigInt(9007199254740991);
// ↪ 9007199254740991n

const hugeButString = BigInt('9007199254740991');
// ↪ 9007199254740991n

5. 类实例

  • 类实例私有属性 / 类实例私有方法

    class Counter extends HTMLElement {
        #xValue = 0; // 类实例私有属性

        get #x() { return #xValue; } // 类实例私有方法
        set #x(value) { // setter // 类实例私有方法
            this.#xValue = value;
            window.requestAnimationFrame(this.#render.bind(this));
        }

        #clicked() { // 类实例私有方法
            this.#x++;
        }

        constructor() {
            super();
            this.onclick = this.#clicked.bind(this);
        }

        connectedCallback() { this.#render(); }

        #render() { // 类实例私有方法
            this.textContent = this.#x.toString();
        }
    }
    window.customElements.define('num-counter', Counter);
