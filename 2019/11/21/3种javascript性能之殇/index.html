3种javascript性能之殇

Posted on 2019-11-21

1. 数组循环

  • for
  • forof
  • while
  • forEach
  • reduce

  • map

    reduce 使用方便

reduce和forEach要求递归调用该函数以递归方式调用并膨胀堆栈，并且对执行的代码进行额外的操作和验证

for while 性能好

2. 复制数组

  • spread operation […arr] 性能中

  • Array from Array.from(arr) 性能最差

  • map arr.map(x => x) 性能次

  • slice arr.slice() 性能好

  • concatenate [].concat(arr) 性能好

3. 迭代对象

  • forin 性能好

  • Object.keys() forEach 性能次

  • Object.entries() forof 性能差
