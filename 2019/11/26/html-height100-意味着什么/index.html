html-height100%意味着什么?

Posted on 2019-11-26

浏览器 html body

首先要确定html不代表浏览器 只是渲染树的最顶层的节点 是不是根节点是一个疑问

html {background: red}

这段代码就证明了 上面的话 你会看见浏览器是背景色是红色
但是实际上html宽度是100% height是0px 但是背景色是红色 这就是一个悖论
高度是0 为何能显示背景色是红色
由此 可以推测 html上面还有根节点 就是浏览器
浏览器默认使用html的背景色渲染

height:100%

html {background: red; height: 100%}
body {background: green;}

背景色还是红色 但是此时是html的背景色  因为html现在高度继承浏览器的高度
拖动就可以证明

html {background: red; height: 100%}
body {background: green;height: 100%}

上面就意味着 html继承浏览器的高度 body继承了html的高度
背景色是绿色

进一步
html
    body
        div#container

html {background: red; height: 100%}
body {background: green;height: 100%}
#container {background: blue, height: 100%}

现在显示是蓝色 #container的高度刚好是浏览器的高度
这就是 html继承浏览器的高度  body继承了html的高度  #container继承body的高度

总结

html默认是没有高度 height100%就会继承浏览器的高度
